# Content Writer Agent

De agentinstructies staan in:

```text
.agents/content-writer.md
```

Gebruik deze agent voor:

- webpagina's herschrijven;
- nieuwe landingspagina's;
- kennisbankartikelen;
- SEO-titels en meta descriptions;
- CTA's en microcopy;
- vertaling van de brand strategy naar concrete websitecopy.

Basisdocument:

```text
docs/brand-strategy.md
```

Korte opdrachtformule:

> Gebruik de Content Writer Agent en schrijf vlotte, zakelijke Cyphers-copy op basis van `docs/brand-strategy.md`.
